Super Puppy

Turn your Mac into a managed local AI server. LLMs, vision, image generation, transcription, TTS, embeddings — controlled from the menu bar.

# clone and run the interactive installer
git clone https://github.com/jerrytalton/super-puppy.git ~/super-puppy
cd ~/super-puppy && ./install.sh
Requires Apple Silicon Mac (M1+) with 64 GB+ unified memory. macOS only.

Menu bar control

Start, stop, and monitor Ollama, MLX, and MCP services. Switch model profiles. See memory usage at a glance.

Standard APIs

OpenAI-compatible endpoints for every model. Your existing code works without changes.

MCP tools for Claude Code

16 tools — generate, review, vision, image gen, transcription, TTS, translation, embeddings, and more — all running on your hardware.

Server & client

Install on a desktop to serve models over Tailscale. Install on a laptop to auto-discover the desktop and fall back to local when away.

Web playground

Test any capability from a browser. Works as a PWA on your phone — your mobile device becomes a frontend to your desktop GPU.

Auto-update

Tagged releases are checked every 2 minutes. Updates apply automatically with crash rollback. install.sh --uninstall for clean removal.
